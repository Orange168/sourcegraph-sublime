Sourcegraph Sublime
===================

  Thank you for installing Sourcegraph Sublime!

  To begin using Sourcegraph Sublime, open any Go file and select a symbol, then press ctrl + alt + j.



  To toggle browser auto-opening, please visit https://github.com/sourcegraph/sourcegraph-sublime#auto-process-selection.
  
  For more details about usage, please visit: https://github.com/sourcegraph/sourcegraph-sublime.
